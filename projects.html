Digital Camera

I created my own digital camera using a webcam, battery pack, keypad, and a Raspberry Pi 3B+. I used fswebcam and a plethora of libraries in order to create a script that allows me to create a functional digital camera with a retro pixelated mode, outdoors mode, and normal mode. I control the camera's interface with the keypad and use the flash on top of the webcam to determine when the process of taking a photo has been completed. I power the camera with a battery pack I had found inside of my house. The python script runs off of boot by creating a systemd service file.

Docketing Tool

One of my tasks during my internship was to automate the docketing process. Docketing referred to documenting the time taken per task in the form of an Excel file that is parsed through and a rate of pay per task is determined. For this project, I created a GMail to Microsoft Excel Python script that elevated productivity and reduced workload by 25%. It has three search fields, a gmail account finder, key word finder, and a date range finder. This allowed for precise searching to occur and the specific time periods of docketing to be processed. After the search fields are inputted, the data is parsed and formatted into an Excel document that is downloaded onto the users computer. It has a .exe file and as such there is no set up process.
